## Installation

Prepare virtual env:

    python3.9 -m venv .venv

Activate venv:

    . .venv/bin/activate

Install dev depencencies:

    pip install -r requirements_dev.txt

## Run

Execute the module you prefer:

* Binary search

      python -m algorithms.binary_search

* Binary tree

      python -m algorithms.binary_tree
